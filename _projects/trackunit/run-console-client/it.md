## Run Console Client
>Supporta TLS.

Dalla root del repository `client-console`:
```bash
dotnet run
```

- Segui il prompt di MSAL device-code.  
- Usa la console per richiedere un'analisi e osservare gli aggiornamenti in tempo reale.
